## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/paulorenan/app-perguntas.git
```

Entre no diretório do projeto

```bash
  cd app-perguntas
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
